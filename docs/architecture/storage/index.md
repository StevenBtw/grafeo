# Storage Model

Grafeo uses a hybrid storage model optimized for graph workloads.

## Overview

```mermaid
graph TB
    subgraph "Logical Layer"
        NODES[Nodes]
        EDGES[Edges]
        PROPS[Properties]
    end

    subgraph "Physical Layer"
        COLS[Columnar Storage]
        ADJ[Adjacency Lists]
        IDX[Indexes]
    end

    subgraph "Persistence Layer"
        WAL[Write-Ahead Log]
        DATA[Data Files]
    end

    NODES --> COLS
    EDGES --> ADJ
    PROPS --> COLS
    COLS --> DATA
    ADJ --> DATA
    IDX --> DATA
    WAL --> DATA
```
