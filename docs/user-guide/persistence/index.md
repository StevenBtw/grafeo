# Persistence

Graphos supports multiple storage modes for different use cases.

## Storage Modes

| Mode | Durability | Performance | Use Case |
|------|------------|-------------|----------|
| In-Memory | None | Fastest | Testing, temporary data |
| Persistent | Full | Fast | Production workloads |
